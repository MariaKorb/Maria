Pergunta

Três pessoas estão em um bar, José, Maria e João José ta olhando para a Maria, mas a Maria ta olhando pro João. o José é casado e o João é solteiro. Com base nestas informações pode-se afirmar que uma pessoa casada está olhando para uma pessoa solteira? a)Sim b)Não c)Não se pode chegar a uma conclusão

Hipótese 1

José é casado, Maria Solteira e João Solteiro.

CASADO ----> SOLTEIRA -----> SOLTEIRO

Hipótese 2

José é casado, Maria casada e João solteiro.

CASADO ----> CASADA -----> SOLTEIRO

Conclusão


Depois de analisar as duas hipóteses é possível perceber que a resposta correta é a A, pois independende da Maria ser ou na casada, é possível afirmar que uma pessoa casada está olhando para uma solteira. Se Maria for solteira, José(que é casado) está olhando para ela, logo uma pessoa casada está olhando para uma solteira. Se Maria for casada, como ela está olhando para João(que é solteiro), logo, pode-se afirmar que uma pessoa casada está olhando para uma solteira.
